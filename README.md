# Factor Fund (showcase)

**Goal**: Publicly document the research & production process of a factor-based strategy while keeping proprietary code, parameters and raw data private.

## Pipeline (high-level)
- Stage 01: Ingestion & Source Catalog
- Stage 02: Calendars, FX & Normalization
- Stage 03: Corporate Actions & Price Adjustments
- Stage 04: Advanced Cleaning (outliers & gaps)
- Stage 05: Base Features & Lags
- Stage 06: Train/Valid/Test Splits & Cross-Validation
- Stage 07–11: Consolidation → Strategies → Fees → Charts → Metrics

> The original internal documentation was translated for public release.
